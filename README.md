# ClassOrbit
ClassOrbit is a university room booking platform that lets students, teachers, and clubs reserve rooms for study sessions, group work, makeup classes, or events. It shows real-time availability, simplifies scheduling, and ensures efficient use of campus spaces while providing an easy and intuitive booking experience.
---

## Features

- **Student Management:** Add, edit, and manage student records easily.
- **Class Management:** Organize classes, assign teachers, and track schedules.
- **Attendance Tracking:** Monitor attendance with real-time updates.
- **Exam & Grading System:** Record marks, generate report cards, and manage results.
- **Dashboard:** Overview of students, classes, and activities.
- **Notifications:** Alerts for upcoming exams, events, or announcements.
- **Secure Login System:** User authentication for teachers and administrators.
- **Admin Panel:** Full control over students, teachers, classes, and system settings.
- **Analytics & Reports:** Generate insights for performance tracking.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.
- **Backup & Restore:** Maintain secure backups of your data.
